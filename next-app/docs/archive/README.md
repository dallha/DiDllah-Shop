# Archive

Ce dossier contient les documents archivés du projet DiDallah Shop :

- `decisions/` — ADR (Architecture Decision Records) : décisions techniques documentées
- `postmortems/` — Analyses post-incident pour les incidents P0/P1
- `ideas/` — Idées de fonctionnalités en cours d'évaluation
- `versions/` — Anciennes versions des documents de gouvernance

## Convention de nommage

```
decisions/  ADR-001-choix-supabase.md
            ADR-002-zustand-vs-context.md

postmortems/ 2026-05-18-quota-exceeded.md

ideas/       idea-variantes-produits.md
             idea-paiement-en-ligne.md
```
